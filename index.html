A* Pathfinding Visualizer

Designed a grid-based map using Java Swing Library, where each cell represents a specific location. Some cells might be walkable, while others could represent obstacles or restricted areas. On the grid, users can specify a beginning point, indicated in grey, and an ending point, indicated in black. The objective is to avoid obstacles and find the shortest route between the beginning and finishing points.
